## 🧭 1. 현재 버전 확인
먼저, PowerShell이 이미 설치되어 있는지 확인하세요.

```powershell
$PSVersionTable.PSVersion
```

- `7.x.x` → 최신 PowerShell (PowerShell Core)
- `5.x.x` → Windows 기본 PowerShell (구버전)

---

## 🪟 2. Windows에서 PowerShell 7.5.4 설치 방법

### ✅ 방법 1: **winget (가장 간단하고 추천)**
Windows 10 이상이라면 기본 내장된 **winget**으로 설치 가능합니다.

```powershell
winget install --id Microsoft.Powershell --source winget
```

> 🔹 이미 설치되어 있다면 최신 버전으로 자동 업데이트됩니다.  
> 🔹 설치 후, 경로는 일반적으로  
> `C:\Program Files\PowerShell\7\pwsh.exe` 입니다.

---

### ✅ 방법 2: **MSI 설치 파일 직접 다운로드**

1. [공식 GitHub PowerShell 릴리스 페이지](https://github.com/PowerShell/PowerShell/releases) 열기  
2. 최신 버전(예: `v7.5.4`) 클릭  
3. 자기가 사용하는 시스템에 맞는 설치 파일 다운로드  
   - 64비트 Windows → `PowerShell-7.5.4-win-x64.msi`
4. MSI 실행 후 기본 옵션으로 설치 진행

> 설치 후 PowerShell을 다시 열면 새 버전이 적용됩니다.

---

### ✅ 방법 3: **업데이트(기존 설치된 7.x 버전이 있을 때)**

```powershell
winget upgrade Microsoft.Powershell
```

---

## 🔧 3. 환경 변수 확인

설치가 완료되면 PowerShell 경로가 자동 등록되어야 합니다.  
확인 명령어:

```powershell
where pwsh
```

> 결과 예시:
> ```
> C:\Program Files\PowerShell\7\pwsh.exe
> ```

---

## 🧩 4. Visual Studio Code와 연동 확인

VS Code 터미널에서 PowerShell이 제대로 연결되는지 확인하려면:

1. VS Code → **Ctrl+Shift+P**
2. “Terminal: Select Default Profile” 검색
3. “PowerShell” 선택

---

## 🚀 5. (선택) PowerShell을 관리자 모드 기본 실행으로 설정

바탕화면 아이콘 → 오른쪽 클릭 → **속성 → 호환성 → 관리자 권한으로 실행** 체크
